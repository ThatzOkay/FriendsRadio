# FriendsRadio

A Fabric mod for Minecraft 1.20.1 that lets you listen to internet radio stations in-game — either by carrying a radio in your hand or by placing one down as a block.

## Features

- **Radio item & block** — place the radio down in the world, or hold it in either hand to take your music with you.
- **Station browser** — search and tune in to internet radio stations, plus save your own favorites and custom stations.
- **Note particles** — a placed or held radio that's playing emits musical note particles so you can spot it from a distance.

## Requirements

- Minecraft 1.20.1
- [Fabric Loader](https://fabricmc.net/use/) 0.19.3 or newer
- [Fabric API](https://modrinth.com/mod/fabric-api)
- [Fabric Language Kotlin](https://modrinth.com/mod/fabric-language-kotlin)
- Java 17+

## Installation

1. Install [Fabric Loader](https://fabricmc.net/use/) for Minecraft 1.20.1.
2. Download and place [Fabric API](https://modrinth.com/mod/fabric-api) and [Fabric Language Kotlin](https://modrinth.com/mod/fabric-language-kotlin) into your `mods` folder.
3. Download FriendsRadio and place it in the same `mods` folder.
4. Launch the game using the Fabric profile.

## Usage

- **Holding the radio**: press `R` to open the station menu and pick something to listen to.
- **Placed radio**: right-click the block to open the same menu.
